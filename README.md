# TimberShop

A premium, beginner-focused woodworking website built with Next.js and Tailwind CSS.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
npm run start
```

## Deploy to Vercel
1. Create a new Git repository and push this project.
2. Import the repository into Vercel.
3. Accept the default framework detection for Next.js.
4. Deploy.

## Notes
- The email signup is currently visual/demo only.
- The site is structured as a content-first single-page experience with anchored navigation.
- Components are organized so future expansion into separate routes is straightforward.
